Add a floating WhatsApp click-to-chat button so visitors can message you in one tap.

== Description ==

Put a friendly floating WhatsApp button on every page. Visitors tap it and land
in a chat with you, optionally with a pre-filled message. Set your number,
choose a side, and you're live — no code.

**Free features**

* Floating button, bottom-left or bottom-right.
* Pre-filled message.
* Icon-only or labelled button.

**Pro features**

* Show only during business hours.
* Multiple agents / departments.
* Priority support.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate.
3. Go to Settings → WhatsApp Chat Button, enter your number, tick "Show".

== Frequently Asked Questions ==

= What number format do I use? =

International format, digits only, including country code — e.g. 14155552671.
